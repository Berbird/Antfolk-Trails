
# Antfolk Trails

## Inspiration

Our team loves learning about nature. We thought people would like to learn about it too, so we decided to research some animals. We found that ants would be interesting and suitable for a game. We're also huge fans of AntsCanada, which gave us our first spark of inspiration.

## What It Does

We play as a springtail that disguises itself as an ant using pheromones. There is a symbiotic relationship between ants and springtails. This game teaches players what ants eat, gather, and how their colonies are shaped.

## How We Built It

- **Engine:** Godot Engine 4
- **Version Control:** Github Repository via Github Desktop
- **Art & Animation:** All assets created in Aseprite
- **Audio:** Sound effects and music from Pixabay

## Challenges We Ran Into

Since there aren't many new Godot tutorials, our code often didn't work properly. We learned GDScript from scratch and still faced many troubles. Additionally, conveying information through a game was difficult—we needed to carefully plan how to balance education with gameplay.

## Accomplishments We're Proud Of

- Debugged and tested all code extensively, fixing every bug we encountered
- Completed the game as planned
- Learned GDScript from scratch
- Created an impressive visual design with thoughtful animations
- Developed new features independently without outside instructions
- Successfully created an educational ant game that teaches players about ant lifestyles

## What We Learned

We learned that teaching with tools to make learning fun can be hard, but it's incredibly rewarding. We also gained proficiency in tools and programs like Godot, Github, and Aseprite.

## 🚀 What's Next for Antfolk Trails

We plan to add new colonies featuring ants from around the world, including:

- Cephalotes
- Leafcutter Ants
- Weaver Ants

We'll add different habitats for each ant type so players can learn about the many ant species around them.
